# MAICA-MTTS Synbrace

Synbrace(次元呼唤) 是 MAICA-MTTS 的前端, 基于 MAICA-MTTS 的文字声学子模组.

本页面的介绍是十分简短的. 如果这是你首次接触 MAICA, 请阅读介绍页 https://maica.monika.love/ 中的详细说明.

> 注意: 需要一个 Doki Chinese Club 账号且必须验证邮箱才能使用, 具体要求请查看[TOS](https://maica.monika.love/tos).  
> 游戏内也会对此提供必要的引导.

## 特色

* 全面的TTS支持, 兼容原版/MAICA/多数其它Submod对话
* 充分融入原版的外围剧情和场景设计
* 表现经过充分的训练, 评估和调整
* 安装简单, 无须在自己的设备上部署任何模型
* 高可自定义化的参数设置
* 开源和二次开发友好
* 仍在积极开发和维护中
  
## 安装

1. 确保使用的是最新汉化版本MAS.  
2. 从[Release](https://github.com/Mon1-innovation/MAICA_MttsSubmod/releases)处下载最新的版本.
  > 该子模组依赖在线服务运行, 旧版本会被弃用失效. 请保持使用最新版本.  
  > 该子模组对MAICA Blessland提供兼容, 但也可以不安装Blessland独立工作.
3. 关闭游戏, 将zip中的文件合并到您的`DDLC`/`MAS_CN001***/Monika After Story`文件夹内, 或者是`DDLC.exe`/`MAS.exe`所在的位置
  > 你不能直接将文件解压至Submods文件夹! 打开压缩包, 并根据文件夹结构合并.
4. 目前不建议通过`Submod Updater Plugin`更新, 因为该子模组的发行版包含actions生成的内容.

# 提示
1. 完整的功能只会在好感度不低于0(NORMAL)时开放.
2. 安装或达到条件后, 相关对话不会立刻解锁, 请遵循游戏内外流程中的指示.
3. 提供对中英双语的完整支持, 但模型在中文上的表现可能较弱.
4. 不支持http(s), socks系统代理. 在系统的设置中将其关闭.
5. 你可以删除`game\Submods\MAICA_MttsSubmod\donation`文件以禁用捐赠链接.
